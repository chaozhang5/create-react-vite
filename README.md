## 介绍

create-react-vite 脚手架工具，基于 React + Vite + TS + Ant + ESLint + Prettier + Stylelint + Husky

## Git 仓库地址

## 启动

```bash
# 安装依赖
yarn install

# 启动服务
yarn dev
```

## 发布

```bash
# 构建开发环境
yarn build:dev

# 构建测试环境
yarn build:test

# 构建生产环境
yarn build:prod
```

## 规范

```bash
# eslint
yarn lint

# stylelint
yarn lint:style

# prettier
yarn format
```
